# docker-teamspeak
Image based on Linux Alpine with Teamspeak installed.

[![GitHub](https://img.shields.io/badge/GitHub-master-green)](https://github.com/moritz-topp/docker-teamspeak)
[![DockerHub](https://img.shields.io/badge/DockerHub-latest-blue)](https://hub.docker.com/r/moritztopp/docker-teamspeak)

## Usage
The App will auto-install the lastest Version in the /app folder,
your data can be stored in the /data folder when mounted.

Images will be updated weekly through Github Actions.
